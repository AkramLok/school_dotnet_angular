Title: Git Mastery Online Course: Certify Your Expertise


What you will learn
  • Learn the key concepts of the Git version control system
  • Navigate through the entire Git workflow
  • Compare different states in Git
  • Manage files inside and outside of Git
  • Create and manage repositories on GitHub
  • Manage local and remote repositories with Git

Description:

This course is designed to distill academic theory into key concepts and focus on basic tasks in Git to make you productive


For whom is this course intended?
  • Anyone interested in source control, particularly Git.
  • Software engineers, developers, and programmers new to Git.
  • IT managers or technical leads considering Git for version control within their team.
